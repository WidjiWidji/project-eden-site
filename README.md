# Project Eden — public showcase site

Static vision + hardware pages for **Project Eden**, an autonomous indoor grow
system (one wall plug, three modules, an AI brain). Served via GitHub Pages:
**https://widjiwidji.github.io/project-eden-site/**

This is a published mirror of `docs/site/` from the private engineering repo —
edit it there, not here. Diagrams are generated drafts (block schematics +
isometric housing previews), for design review and presentation, not fabrication.
